# 🏋️‍♂️ Fitness Sports Center Website

A modern and responsive **promotional landing page** built for a fitness brand as part of a **Web Development Internship evaluation**. The website showcases gym services, membership plans, trainers, and includes a functional contact form for user inquiries.

## 🚀 Features

- 🖥️ Fully responsive design (mobile, tablet, desktop)
- 🎯 Clean and modern UI based on provided branding
- 🌙 Dark mode with theme persistence
- 🧩 Reusable React components (Card, Navbar, Sections)
- 🏋️ Hero section with image blending effect
- 📊 Services and membership plans display
- 👨‍🏫 Trainers showcase section
- 📬 Contact form with:
  - Input validation
  - Success feedback message
- ⚡ Smooth scrolling navigation
- 🎨 Hover effects and interactive UI elements

---

## 🛠️ Tech Stack

- **React.js (Vite)**
- **Tailwind CSS v4**
- **JavaScript (ES6+)**
- **React Icons**

---

## 🧪 How It Works

- The application is built using **component-based architecture** in React.
- Sections like Hero, About, Services, Plans, Trainers, and Contact are modular and reusable.
- Theme switching (light/dark mode) is handled using state + `localStorage`.
- UI colors are managed using **CSS variables** for easy theme control.
- The contact form uses **custom validation logic** and displays real-time error messages and success feedback.

## 📦 Installation

```bash
# Clone the repository
```
